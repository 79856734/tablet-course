HTML (لغة توصيف النص الفائق)
- الغرض:
HTML هي اللغة الأساسية لتوصيف وترتيب المحتوى على الويب. بتوفر الهيكل الأساسي لصفحة الويب وبتنظم العناصر زي النصوص، الصور، الروابط، القوائم، والنماذج.

- البنية:
مستندات HTML بتتكون من عناصر متداخلة أو وسوم. كل وسم بيحدد نوع معين من المحتوى:

○ html: العنصر الجذري اللي بيغلف المستند كله.
○ head: بيحتوي على معلومات عن المستند زي العنوان، مجموعة الأحرف، وربط أوراق الأنماط والسكريبتات.
○ body: بيلف المحتوى الظاهر للصفحة.
○ الوسوم زي h1 لـ h6 (عناوين)، p (فقرات)، img (صور)، a (روابط)، div (أقسام)، و span (عناصر داخلية) بتستخدم لترتيب المحتوى.

CSS (أوراق الأنماط المتتالية)
- الغرض:
CSS بتتحكم في عرض وتنسيق وتخطيط عناصر HTML. بتفصل المحتوى عن التصميم، وده بيسهل تحديث المظهر البصري لصفحات الويب.

طريقة العمل:
○ المحددات (Selectors): CSS بتستخدم محددات (زي عناصر، كلاسات، ومعرفات) لاستهداف عناصر HTML.
○ الخصائص والقيم: كل محدد بيتحدد له خصائص (زي اللون، حجم الخط، الهوامش، والحشو) بقيم معينة.
○ الميزات المتقدمة: CSS بتدعم الحركات، الانتقالات، فليكس بوكس، تخطيطات الشبكة، والخصائص المخصصة (المتغيرات) لخلق تصميمات ديناميكية ومرنة.

JavaScript (JS)
الغرض:

JavaScript هي لغة برمجة بتضيف التفاعلية والمحتوى الديناميكي والميزات المعقدة لصفحات الويب. بينما HTML بيبني الهيكل وCSS بتضبط المظهر، JavaScript بتخلي الموقع ينبض بالحياة عن طريق تمكين السلوك التفاعلي.

- الوظائف:
○ التعامل مع DOM: JS بتقدر تتفاعل مع نموذج كائن المستند (DOM) لتعديل أو إضافة أو حذف عناصر HTML في الحال، وده بيسمح بتحديث المحتوى بدون إعادة تحميل الصفحة.
○ معالجة الأحداث: JavaScript بتسمع لأحداث المستخدم (زي النقر، ضغط المفاتيح، وحركة الماوس) وبترد عليها، مما بيدي تجربة تفاعلية.
○ التعامل مع البيانات وAPIs: JS بتقدر تجيب وتبعت بيانات للسيرفرات باستخدام AJAX أو Fetch API، وده بيسمح بتحديث المحتوى ديناميكيًا والتفاعل مع خدمات الخلفية.
○ الأطر والمكتبات: التطوير الحديث بيستخدم أطر عمل ومكتبات (زي React، Angular، أو Vue) اللي بتبني على JavaScript لتسهيل عملية التطوير وإدارة التطبيقات المعقدة.
○ تحسين تجربة المستخدم: ميزات زي التحقق من صحة النماذج، الخرائط التفاعلية، الألعاب، ومشغلات الوسائط بقت ممكنة بفضل JavaScript.

إزاي بيشتغلوا سوا
- HTML بيوفر هيكل الصفحة.
- CSS بتضبط المظهر والتنسيق.
- JavaScript بتضيف السلوك، وبتخلي العناصر ترد على إجراءات المستخدم وتتفاعل بشكل ديناميكي.

على سبيل المثال، لما تدوس على زر (عنصر HTML)، JavaScript ممكن تغير شكله (عن طريق CSS) أو تحدث محتوى قسم بدون ما تعيد تحميل الصفحة كلها.

مع بعض، التقنيات التلاتة دي بتمكن المطورين من بناء تطبيقات ويب قوية وتفاعلية وجذابة بصرياً.
